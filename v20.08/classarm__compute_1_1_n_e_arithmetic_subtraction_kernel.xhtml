Initialise the kernel's input and output.

Valid configurations (Input1,Input2) -> Output :

  • (U8,U8) -> U8
  • (U8,U8) -> S16
  • (QASYMM8, QASYMM8) -> QASYMM8
  • (QASYMM8_SIGNED, QASYMM8_SIGNED) -> QASYMM8_SIGNED
  • (S16,U8) -> S16
  • (U8,S16) -> S16
  • (S16,S16) -> S16
  • (F16,F16) -> F16
  • (F32,F32) -> F32
Parameters
[in]input1An input tensor. Data types supported: U8/QASYMM8/QASYMM8_SIGNED/QSYMM16/S16/F16/F32
[in]input2An input tensor. Data types supported: U8/QASYMM8/QASYMM8_SIGNED/QSYMM16/S16/F16/F32
[out]outputThe output tensor. Data types supported: U8/QASYMM8/QASYMM8_SIGNED/QSYMM16/S16/F16/F32.
[in]policyOverflow policy. Convert policy cannot be WRAP if datatype is quantized.

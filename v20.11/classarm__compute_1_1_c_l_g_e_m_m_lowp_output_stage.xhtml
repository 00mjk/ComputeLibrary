Detailed Description

Basic function to execute GEMMLowpQuantizeDown kernels on CL.

This function calls the following CL kernels:

  1. CLGEMMLowpQuantizeDownInt32ScaleKernel
  2. CLGEMMLowpQuantizeDownInt32ScaleByFloatKernel
  3. CLGEMMLowpQuantizeDownInt32ScaleByFixedPointKernel
